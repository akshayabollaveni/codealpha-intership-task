Hangman Game

Description

The Hangman Game is a simple Python-based word guessing game. A random word is selected from a predefined list, and the player must guess the hidden word one letter at a time. The game uses Python concepts such as lists, strings, random module, while loops, and if-else statements.

Features

- Random word selection
- Hidden word display using underscores
- 6 guessing chances
- Hangman drawing for wrong guesses
- Win and lose conditions
- Interactive console game

Concepts Used

- Lists
- Strings
- Random Module
- While Loop
- If-Else Statements
- User Input
- Indexing
